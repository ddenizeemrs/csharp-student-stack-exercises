# C# Student Stack Exercises

A university coursework assignment consisting of two C# console applications focused on basic data structures and control flow.

## Program 1

The application collects the following information for three students:

- First name
- Last name
- Age

The data is stored in an array and displayed in reverse order without using a Stack structure.

## Program 2

The application asks the user how many student records will be entered.

Each student's information is stored in a `Stack<string>` structure and later displayed in reverse order using the LIFO principle.

## Concepts Used

- C# Console Applications
- Arrays
- Loops
- User Input
- Stack
- Push and Pop Operations
- LIFO Data Structure

## Technologies

- C#
